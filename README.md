# Team Titans (E-Waste Hub)

![Image](./image.png)

#### How to run

##### To Run Client

    cd client

    npm install

    npm run start

##### To Run Server

1.  Install flask

    ```
    pip3 install Flask

    ```

2.  create and activate python environment

    ```
    cd server

    python3 -m venv .venv

    // in windows
    .venv\Scripts\activate

    // in mac
    source .venv/bin/activate
    ```

3.  run the flask app

    ```
    flask --app app run
    ```
